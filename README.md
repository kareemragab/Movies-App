# Movies-App
App that displays a list of movies fetched through a web API and allows users to add their own.

# Run on android 

cd Movies-App <br />
yarn install <br />
npx react-native start <br />
npx react-native run-android <br />
 
# Run on ios 
cd Movies-App <br />
yarn install <br />
npx react-native start <br />
npx react-native run-ios
